# Release Notes - Version 2.5.0

## [2.5.0] - 2026-02-02

### Added
- **[cli]** command `wiz project` added
    - `build`: Build project (supports cache cleaning and automatic app.json update)
    - `create`: Create new project from template
    - `delete`: Delete project
    - `list`: List projects
    - `clean`: Clean project cache
- **[cli]** command `wiz project app` added
    - `list`: List apps in project
    - `create`: Create new app (supports engine selection: html/pug)
    - `delete`: Delete app
- **[cli]** command `wiz project controller` added
    - `list`: List controllers
    - `create`: Create new controller
    - `delete`: Delete controller
    - `update`: Update controller content

### Changed
- **[core]** `wiz.ide.plugin.model` and `command` methods updated to support `mode='context'` for CLI context injection without performance overhead in web mode.
- **[cli]** Refactored CLI architecture for better modularity (core, app, controller modules).
- **[builder]** Improved build process stability (handling missing directories).

### Fixed
- **[cli]** Fixed path resolution for recursive project root finding and symlink handling.
- **[cli]** Fixed dependency injection issues in plugin execution within CLI environment.
